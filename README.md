# wingman

> Initialise projects

## Installation

```sh
yarn global add https://github.com/rajatsharma/wingman
```

## Usage

- Generate Go Project

```sh
wingman go go-project
```

- Generate Rust Project

```sh
wingman rust rust-project
```

- Generate Phoenix Graphql Project

```sh
wingman phoenix-graphql phoenix-project
```
