# wingman

> Open Source management utility

## Usage

- Generate Rust Project

```sh
wingman rust-init rust-project --bin
```

- Generate Purescript Project

```sh
wingman purs-init purs-project
```

- Generate Nextjs Project

```sh
wingman next-init next-project
```
